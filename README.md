# Metro All Time

## Project Overview
Metro All Time is a project designed to provide real-time transit information for metropolitan areas. The application offers features such as:

- Live tracking of public transportation
- Alerts for delays and disruptions
- User-friendly interface for easy navigation

## Technologies Used
- JavaScript
- React
- Node.js
- MongoDB

## Installation
To install the project, clone the repository and run:
```
npm install
```

## Usage
Run the following command to start the application:
```
npm start
```
