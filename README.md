# Metro All Time

## Descripción del Proyecto
**Metro All Time** es un proyecto diseñado para proporcionar información de transporte público en tiempo real para áreas metropolitanas. La aplicaciónesta pensada para ofrecer funcionalidades como:

- Seguimiento en vivo del transporte público  
- Alertas sobre retrasos e interrupciones del servicio  
- Interfaz intuitiva para facilitar la navegación  

## Tecnologías Utilizadas
- JavaScript  
- React  
- Node.js  
- MongoDB  

## Instalación
Para instalar el proyecto, clona el repositorio y ejecuta:

```bash
npm install
```

## Uso
Ejecuta el siguiente comando para iniciar la aplicación:

```bash
npm start
```
